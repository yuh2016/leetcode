##算法

###排序

####选择排序 Selection
>选择最小的元素与第一个元素交换。然后在剩下的元素中选择最小的元素与第二元素交换。。。
#####时间复杂度
- 交换次数：N
- 比较次数：(N-1) + (N-2) + ... + 1 = N(N-1)/2 ~ N^2

####插入排序 Insertion
>每个元素与之前的依次比较，小的话交换

#####时间复杂度
- 交换次数 最小N次，最大 
- 比较次数 最小N次，最大 

###希尔排序 Shell
基于插入排序（插入排序对于小规模、有序的数组很有用）
>使数组中任意间隔为h的数组都是有序的
>最终对于h=1，对数组进行插入排序
>如果h很大，可以使较大的元素直接移动到右侧
